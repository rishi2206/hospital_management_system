# Hospital Management System

A Hospital management system backend is built using FastAPI, PostgresSQL, Docker, 

This project provides to manage the users to manage doctors, patients, Billing and appointments in a Hospital.

## Tech Stack Used

* Python
* FastAPI
* PostgresSQL
* SQLAlchemy Async
* AsyncPG
* Pydantic
* Uvicorn
* Docker
* Docker Compose
* pgAdmin
* postman
* HTTPX
